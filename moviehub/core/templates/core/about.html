About MovieHub

MovieHub is a demo learning project showing how to build a movie information site using Django and the TMDb API. This site fetches movie data (titles, posters, trailers, cast) from TMDb and stores user-specific data like watchlists and reviews in a local database.

Purpose

This project is built for learning web development: Django backend, templates, JavaScript interactions (modal, infinite scroll), and simple persistence for user data.

Credits

  • TMDb — The Movie Database (data & images)
  • Bootstrap — UI framework
  • Font Awesome — Icons
